validateShapeDimensions

  • validateShapeDimensions(): any
    • Defined in index.ts:331

    Ensures that the height & width are less than that of the canvas'. If found to be in excess of that, it will reassign the height & width to the modulus of the input.

    An error message will be displayed to the user notifying them of the change to their square's dimensions, then it will call the parent drawShapeElement() function.
